\# DevConnect



DevConnect is a full-stack web application that allows users to connect through a contact form. It's built using:



\- \*\*Frontend:\*\* React, Tailwind CSS

\- \*\*Backend:\*\* Node.js, Express

\- \*\*Database:\*\* MongoDB
